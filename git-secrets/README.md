# git-secrets

## Why?

* Prevent Service Account JSON Key File accidental commit/upload

## Getting Started

Installation

```bash
./install.osx.git-secrets.sh
```

Bypassing the warning on a `git commit`

```bash
git commit -m "Message" --no-verify
```
